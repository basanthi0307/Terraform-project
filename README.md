## Services Used
- Terraform
- AWS EC2
- VPC
- Subnets
- Security Groups
- Internet Gateway
- Route Tables
- Application Load Balancer
- Target Groups

## Commands

terraform init
terraform plan
terraform apply
terraform destroy

## Output

- Two EC2 Instances
- ALB DNS
- Apache Web Server
- User Data
